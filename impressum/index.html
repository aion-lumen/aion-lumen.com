Data ProtectionDatenschutz

This is a static website. It sets no cookies and embeds no external services. For basic reach measurement we evaluate server access logs only; IP addresses are anonymised (last octet removed) and raw logs are deleted after 7 days. No visitor profiles are created. Dies ist eine statische Website. Sie setzt keine Cookies und bindet keine externen Dienste ein. Zur einfachen Reichweitenmessung werten wir ausschließlich Server-Zugriffsprotokolle aus; IP-Adressen werden dabei anonymisiert (letztes Oktett entfernt) und die Rohprotokolle nach 7 Tagen gelöscht. Es werden keine Besucherprofile erstellt.

The only data stored is a small set of local display settings (theme, language, hand-script) kept in your browser via localStorage. These never leave your device. Gespeichert werden lediglich einige lokale Anzeige-Einstellungen (Theme, Sprache, Hand-Schrift) im Browser via localStorage. Diese verlassen Ihr Gerät nicht.
